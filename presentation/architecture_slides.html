Current Challenges

Data Limitations

  • Small dataset (599 windows)
  • Severe class imbalance (6 vs 216 samples)
  • High dimensionality (3 axes × 200 samples)

Model Limitations

  • Current CNN architecture may be suboptimal
  • Training instability with small batch sizes
  • Overfitting indicators in validation performance

Our current approach using class weights and SMOTE has improved performance, but architecture optimization may yield further gains.

Architecture Search Approach

We propose a systematic neural architecture search to identify the optimal model for whale behavior classification:

Search Space

  • Model family: CNN, RNN, Transformer, Hybrid
  • Layer width: 16-128 units
  • Layer depth: 1-5 layers
  • Regularization: Dropout rates 0.1-0.5
  • Activation functions: ReLU, ELU, GELU

Evaluation Metrics

  • Hierarchical group accuracy
  • Raw classification accuracy
  • F1-score on minority classes
  • Parameter count
  • Inference latency

Current Architecture Analysis

class TCN(nn.Module): def __init__(self, input_size, output_size, num_channels, kernel_size, dropout, parallel=False): super(TCN, self).__init__() self.tcn = TemporalConvNet(input_size, num_channels, kernel_size, dropout=dropout) self.linear = nn.Linear(num_channels[-1], output_size) self.parallel = parallel def forward(self, x): # Input shape: (batch_size, sequence_length, input_size) if self.parallel: # For multi-GPU training x = x.permute(0, 2, 1) else: # x needs to have dimension (batch_size, input_size, sequence_length) x = x.transpose(1, 2) output = self.tcn(x) output = output[:, :, -1] # Take the output of the last time step output = self.linear(output) return output
