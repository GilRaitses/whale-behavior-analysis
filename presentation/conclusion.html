Project Accomplishments

Data Limitation Notice

The results presented are preliminary proof-of-concept findings with significant limitations:

  • Current dataset: ~20,000 frames with only ~50 expert annotations
  • Extreme class imbalance: some behaviors have only 1 example
  • Validation metrics should be interpreted with extreme caution

Despite data limitations, this project established:

  • A complete machine learning pipeline for whale behavior classification
  • Effective dimensionality reduction techniques for visualization
  • Interactive tools for analyzing dive patterns and behaviors
  • Framework for integrating expert annotations with automatic classification
  • Hierarchical visualization approach for multi-level behavior analysis

Data Challenges

The current project faces critical data limitations:

  • Insufficient samples for reliable model training
  • Underrepresented classes preventing proper validation
  • SMOTE and other augmentation techniques cannot fully compensate for fundamental data scarcity
  • Lack of properly validated ground truth data
  • Annotation inconsistencies across limited samples

Primary Data Needs

To establish reliable metrics and models, we need:

  • Minimum 10 expert-annotated examples per behavior class
  • Separate validation dataset with confirmed ground truth
  • Consistent annotation methodology
  • Additional whale subjects to validate generalization
  • Expanded temporal coverage to capture seasonal variations

Future Directions

Data Collection Priorities

  1. Collaborate with Parks Lab to obtain additional expert annotations
  2. Establish a proper validation methodology with separate test dataset
  3. Focus on increasing examples for underrepresented behavior classes
  4. Develop a standardized annotation protocol for consistency

Once data limitations are addressed, future work includes:

  • Refining model architecture based on expanded dataset
  • Improving temporal modeling across extended behavior sequences
  • Developing more sophisticated ensemble methods
  • Exploring behavioral transfer learning across subjects
  • Expanding the interactive dashboard capabilities for researchers
