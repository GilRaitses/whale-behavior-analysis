4. Sliding Window Creation

Role

Segments continuous time series data into fixed-length windows for classification.

Implementation

  • Creates overlapping windows of sensor data
  • Associates each window with the corresponding behavior label
  • Filters out windows with unlabeled data

Impact

The window creation process produced 599 windows of labeled behaviors after filtering unlabeled data.

This step transforms the continuous time series into discrete samples for classification, creating the foundation for the machine learning process.

Window parameters:

  • Window size: 100 time points
  • Overlap: 50%
  • Dimensions: (batch_size, 100, 20)

Windowing is a critical step that transforms the continuous sensor data into a format suitable for deep learning, capturing temporal patterns while maintaining context.

6. Class Weighting

Role

Further addresses class imbalance by making the model more sensitive to rare classes.

Implementation

  • Computes weights inversely proportional to class frequency
  • Ignores unlabeled class (Class 0) by setting its weight to 0
  • Applies weights to loss function during training

8. Model Architecture (minGRU)

Role

Processes temporal patterns in the windowed sensor data to classify behaviors.

Implementation

  • Uses minimal Gated Recurrent Unit (minGRU) architecture
  • Processes fixed-length windows of multi-dimensional time series
  • Outputs classification probabilities for each behavior class

From the logs:

Model parameters: 39690

minGRU Architecture Benefits

  • Efficiency: Fewer parameters than standard GRU
  • Memory retention: Maintains ability to track long-term dependencies
  • Faster training: Reduced computational requirements
  • Reduced overfitting: Especially important for small datasets

The architecture includes:

  • Input layer: 20 features × 100 time points
  • minGRU layer with hidden dimension 128
  • Dropout layer (0.3)
  • Output layer with 10 units (9 behaviors + unlabeled)

The minGRU architecture strikes an optimal balance between model complexity and expressiveness, providing sufficient capacity to learn complex whale behavior patterns while minimizing overfitting risk.
