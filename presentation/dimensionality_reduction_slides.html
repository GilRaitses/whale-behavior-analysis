Feature Transformation Pipeline

Our implementation uses a parallel prefix sum algorithm for efficient temporal feature extraction:

# Parallel prefix sum (scan) algorithm for temporal feature enrichment def parallel_prefix_sum(x, dim=0): """ Compute inclusive prefix sum along specified dimension in parallel """ output = x.clone() n = x.size(dim) # Up-sweep phase (reduction) for d in range(log2(n)): step = 2 ** d indices = torch.arange(0, n, 2 * step, device=x.device) # Create index slices and apply parallel operations gather_indices = indices + step - 1 scatter_indices = indices + 2 * step - 1 # Ensure indices are within bounds valid_indices = scatter_indices < n gather_indices = gather_indices[valid_indices] scatter_indices = scatter_indices[valid_indices] # Parallel gather and scatter output.index_add_(dim, scatter_indices, output.index_select(dim, gather_indices))

Benefits of this approach:

  • Reduces time complexity from O(n) to O(log n) in parallel settings
  • Efficiently computes cumulative statistics across time dimensions
  • Enables extraction of integrated velocity, displacement, and energy features
  • Optimized for Apple Silicon GPU acceleration
